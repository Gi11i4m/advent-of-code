# Advent of Code 2016
My solutions to the Advent of Code programming problems.

Language = TypeScript.

http://adventofcode.com/2016/

## Execution
To set the day to execute
> npm config set advent-of-code:day ${day}

To execute the script for the configured day
> npm start
